# JavaRestAPI
Java Rest API

Tecnologias/Frameworks:
- java version "11.0.8" 2020-07-14 LTS
- Apache Maven 3.6.3
- mysql 5.7.31
- Eclipse
- Swagger
- Postman
- Springboot
- Flyway
- Docker
- AWS
- Unit Test

Setup:
- git clone: https://github.com/cesarmsk/JavaRestAPI.git
- docker-compose up -d --build
- mvn clean package spring-boot:run
